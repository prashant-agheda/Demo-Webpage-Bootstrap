Basic Steps for Git:


  • First Install git software in laptop or pc.

  • Then after installation, open “cmd” or “git bash” that gets installed with git.

  • Type “git --version” to check the version of git and see whether its properly installed or not.

  • Then config your name, email, etc. so that other person can know that who have made changes. There is a config file inside git where we change these all.

  • Type git config --global user.name “Name Surname” and press enter so that your name is set successfully.

  • Now similarly set the email id by git config --global user.email “youremail@gmail.com” and press enter so that your email is set successfully.

  • Then Type git config --global user.name and git config --global user.email and see whether your name and email are getting printed on the next line or not.

  • Other way of changing these things is to type “git config --global --edit” so this will open the file directly and you can make changes into it and save it.

  • This git works with all your projects like whether it is a python file, or java script project, text files, etc. it works with all types of files.

  • Commands: “mkdir directoryName” is used to make a directory and “cd directoryName” is used to change the directory.
